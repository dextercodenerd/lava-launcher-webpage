# Project Overview

This is a very simple webpage for an open source Minecraft launcher called Lava Launcher. It is aimed at kids and families, so it should be very simple and easy to use. It is also open source, so it should be easy to contribute to.

* always write inline HTML and CSS in one HTML file
* do not use any frameworks, just vanilla code
* the launcher is multi-platform (Windows, macOS, Linux)
* do not add hover effect that enlarge items
